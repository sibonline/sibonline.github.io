About SWC

Students Who Code (SWC) is a coding club for students passionate about technology. We focus on programming, game development, robotics, hackathons, and collaborative projects.

SWC hosts workshops to teach coding skills, organizes competitions and game jams, and provides mentorship for students to develop innovative projects.

Activities & Programs

  • Hackathons and coding competitions
  • Robotics challenges and workshops
  • Game jams and collaborative projects
  • Programming mentorship and peer learning
  • Workshops on web development, AI, and software tools

Join SWC

Open to all students interested in coding, technology, and innovation. Members collaborate, learn, and create projects that have real impact.
